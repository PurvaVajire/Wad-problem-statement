PICT B.E. Course Structure

First Year (FE)

  • Engineering Mathematics-I: Fundamental calculus and linear algebra concepts.
  • Basic Electronics: Introduction to electronic components and circuits.
  • Engineering Physics: Mechanics, optics, and materials science.
  • Programming in C: Basics of programming and problem-solving using C.

Second Year (SE)

  • Data Structures: Concepts of stacks, queues, trees, and graphs.
  • Digital Electronics: Logic gates, flip-flops, and digital systems.
  • Computer Organization: CPU, memory, and instruction cycles.
  • Object-Oriented Programming: Classes, inheritance, and polymorphism in C++/Java.

Third Year (TE)

  • Operating Systems: Processes, threads, and memory management.
  • Database Management Systems: SQL, ER modeling, and transactions.
  • Computer Networks: OSI layers, protocols, and network devices.
  • Software Engineering: SDLC, Agile, and software quality assurance.

Final Year (BE)

  • Machine Learning: Supervised, unsupervised learning and models.
  • Cloud Computing: Cloud architecture, IaaS, PaaS, SaaS models.
  • Project Work: Development of a major project with documentation.
  • Electives: Cybersecurity, IoT, Data Mining, etc., based on interest.
